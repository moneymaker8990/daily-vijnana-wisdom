# Google Play Feature Graphic

Required before Google Play Store submission.

## Specifications

- **Dimensions:** 1024 x 500 pixels
- **Format:** PNG or JPEG (PNG preferred for text clarity)
- **Max file size:** 1 MB
- **Orientation:** Landscape

## Content Guidelines

- App name "MindVanta" should be prominent
- Subtitle or tagline: "Ancient Wisdom for Modern Seekers"
- Clean, calm aesthetic consistent with the app's spiritual/wellness theme
- Avoid small text -- the graphic appears at various sizes on Play Store
- Do not include device frames or screenshots (use the screenshot slots for those)
- Safe zone: keep key content within the center 80% to avoid cropping on different layouts

## Where It Goes

- Google Play Console > Store listing > Feature graphic
- Also displayed at the top of the app's Play Store page on some devices

## Output Path

Save the final file as `store-assets/feature-graphic.png` (or `.jpg`).

## Tools

- Canva, Figma, or any design tool
- The existing `public/icon.svg` can serve as a base for the app icon element
