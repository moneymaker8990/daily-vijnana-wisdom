# MindVanta Review Evidence Pack

Use this file when filling App Store Connect review notes and Google Play app content/data safety disclosures.

## 1) Permission Justification

### Notifications
We use notifications to deliver the user's daily wisdom reminder at their chosen time and to celebrate streak milestones. Notifications are optional and can be disabled in app settings.

### Network Access
Network access is used for optional cloud sync and AI dream interpretation when the user explicitly requests it. Core reading/journaling flows continue to work locally.

## 2) AI-Generated Content Disclaimer

Dream interpretations are generated by an AI model and are provided for reflection and entertainment only. They are not medical, psychological, or therapeutic advice, and should not be treated as diagnosis or treatment guidance.

## 3) External Services and Data Handling

- **Supabase**: Optional authentication and cloud sync; Edge Function endpoint for dream interpretation orchestration.
- **Anthropic API**: LLM provider used for dream interpretation responses.
- **Sentry**: Crash/error monitoring for app reliability (non-PII technical errors only).
- **Data sales policy**: MindVanta does not sell personal or health-related data.

## 4) Reviewer Access

- **Core app access**: No login required for core daily wisdom, study, and local journaling features.
- **Optional sign-in features**: Cloud sync and account-based features require user-authenticated access.
- **Demo account**: Core review can proceed without login. If reviewer requests auth path testing, provide temporary credentials in App Store Connect review notes at submission time.
